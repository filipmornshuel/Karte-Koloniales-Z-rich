Karte Koloniales Zürich

Ein digitaler Stadtrundgang durch Zürich

Anleitung

1. Station finden

Auf der Karte weiter unten können Sie verschiedene Stationen erkundschaften

2. Station anschauen

Wenn Sie eine Station gefunden haben, können Sie einfach auf eine Klicken und die Station einfach geniessen

3. Liste der Stationen anschauen

Sie könenn auch optional die Liste der Stationen weiter unten anschauen. Wenn Ihnen das Bild zu klein erscheint, können Sie auf das Bild klicken und so vergrössern. Wenn Sie nochmals klicken, wird sich das Bild wieder schliessen.

4. Eigene Station vorschlagen

Falls Sie selber in Zürich unterwegs sind und eine Kolonialgeschichte zu einer Station gefunden haben, können Sie diese uns einfach vorschlagen, indem Sie den Knopf unten rechts "Station vorschlagen" klicken und das Formular ausfüllen und auf Karte mit dem jeweiligen Standort klicken. Ihre Station wird aber nicht direkt erscheinen, sondern wir werden diese zuerst überprüfen, falls alles korrekt ist, wird Ihre Anfrage angenommen und die Station wird auf der Karte freigeschaltet und für jeden einsehbar.
