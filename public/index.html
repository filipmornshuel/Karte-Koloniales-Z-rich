Anleitung

1. Station finden

Auf der Karte weiter unten können Sie verschiedene Stationen erkundschaften

2. Station anschauen

Wenn Sie eine Station gefunden haben, können Sie einfach auf eine Klicken und die Station einfach geniessen

3. Eigene Station vorschlagen

Falls Sie selber in Zürich unterwegs sind und eine Kolonialgeschichte zu einer Station gefunden haben, können Sie diese uns einfach vorschlagen, indem Sie den Knopf unten rechts "Station vorschlagen" klicken und das Formular ausfüllen.
